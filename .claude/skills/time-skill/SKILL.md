# Time Skill

## Description
Display the current date and time in Pakistan Standard Time (PKT, UTC+5).

## Timezone
- Asia/Karachi (UTC+5)

## Format
- 24-hour format
- Include date with time
- Output format: YYYY-MM-DD HH:MM:SS PKT

## Implementation
Execute bash command:
```bash
TZ='Asia/Karachi' date '+%Y-%m-%d %H:%M:%S %Z'
```

## When to Use
Invoke this skill when users request:
- Current time in Pakistan
- Time in PKT timezone
- Specific Pakistan time queries
