# Time Command

## Description
Display the current time in Pakistan Standard Time (PKT, UTC+5).

## Instructions
Execute the following bash command:
```bash
TZ='Asia/Karachi' date '+%Y-%m-%d %H:%M:%S %Z'
```

## Output Format
Present the result as: "Current Time in Pakistan (PKT): YYYY-MM-DD HH:MM:SS PKT"

## Requirements
- Always use the `Asia/Karachi` timezone (UTC+5)
- Use 24-hour format
- Include the date with the time
- Keep output brief and straightforward
